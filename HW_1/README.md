## Запуск

### 1. Окружение

Создайте `.env.local` файл в корневой папке проекта и поместите в него:
```
VITE_OMDB_API_KEY=741a105
```

### 2. Установка зависимостей и запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `localhost:5173` (адрес по дефолту)

ВАЖНО: поиск фильмов работает исключительно на латинице
